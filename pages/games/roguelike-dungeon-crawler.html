[WIP] Roguelike RPG Dungeon Crawler


This game is a roguelike RPG dungeon crawler made with Godot. It is still a work in progress, but the core gameplay is already implemented.

Features

Many features implemented stem from the Human VS Orc C++ project I made, with the goal of having a modular RPG code architecture.

Currenly implemented features are:

  • Unit attributes;
  • Equipment and usable items;
  • Skills, buff/de-buff and status effects;
  • Turn-based combat;
  • Loot and inventory;
  • Zone progression.

This game may be published on Steam and the Play Store in the future.
